# 💰 Smart Personal Finance Management System

## 📌 Project Overview

The **Smart Personal Finance Management System** is a web-based application that helps users track their income and expenses, manage budgets, and analyze their financial habits through visual dashboards and insights.

The system provides an easy interface for users to add transactions, categorize expenses, view analytics charts, and receive alerts when spending exceeds the budget.

This project demonstrates the use of **Frontend Web Development (HTML, CSS, JavaScript)** along with **Data Structures and Algorithms concepts** for financial data processing and analytics.

---

# 🎯 Objectives

* To help users **track income and expenses efficiently**
* To provide **budget management and alerts**
* To visualize financial data using **interactive charts**
* To generate **AI-like insights about spending patterns**
* To create a **user-friendly financial dashboard**

---

# 🚀 Features

## 🔐 Authentication

* Login and Signup System
* Session-based login using local storage
* Secure logout functionality

## 💰 Transaction Management

* Add income or expense transactions
* Categorize transactions (Food, Travel, Bills, Shopping, etc.)
* Delete transactions
* Transaction history table

## 📊 Analytics Dashboard

* Income vs Expense **Doughnut Chart**
* Category-wise expense analysis
* Savings prediction graph
* Financial summary dashboard

## ⚠ Smart Alerts

* Budget limit warning
* Spending analysis insights
* AI-like financial recommendations

## 🔎 Search System

* Search transactions by description
* Instant results display

## 🌙 UI Features

* Dark Mode / Light Mode toggle
* Responsive dashboard layout
* Modern finance dashboard design

## 📤 Report Export

* Export financial data as CSV report

# 📊 Data Structures Used

The system uses several **DSA concepts**:

* **Arrays** → Store transactions
* **Searching Algorithm** → Find transactions quickly
* **Filtering** → Separate income and expense
* **Aggregation** → Calculate totals and savings
* **Dynamic Updates** → Real-time UI updates

# 🖥 How to Run the Project

1. Download the project folder
2. Ensure the following files exist:

   * `index.html`
   * `style.css`
   * `script.js`
3. Open **index.html** in any web browser.

No installation or server is required.
